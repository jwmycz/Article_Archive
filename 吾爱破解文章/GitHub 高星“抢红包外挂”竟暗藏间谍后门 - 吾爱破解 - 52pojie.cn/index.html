  1. 诱饵阶段：你运行抢红包 App，开启了“无障碍权限”。
  2. 潜伏阶段：marsdaemon 启动双进程保活，UYMManager 初始化。

  5. 加密外传：数据交给 e.java（异或加密+GZIP压缩），最后由 k.java（Https POST） 发往 puata.info。
  6. 毁灭痕迹：上报成功后，e.a(File) 递归删除本地缓存的 net_change 文件，仿佛一切都没发生过。
